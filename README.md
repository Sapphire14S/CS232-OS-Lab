## Laboratory Overview

This repository contains practical assignments and implementations related to core Operating Systems concepts including process management, CPU scheduling, synchronization, memory management, deadlocks, file systems, and system-level programming.

---

## Labs Overview

| Lab | Concepts Covered |
|-----|------------------|
| Lab 01 | Introduction to Operating Systems, Linux environment, basic system commands |
| Lab 02 | Process creation, process control, `fork()`, `exec()`, process hierarchy |
| Lab 03 | CPU scheduling algorithms and process scheduling techniques |
| Lab 04 | Inter-process communication (IPC), pipes, shared memory |
| Lab 05 | Process synchronization, semaphores, mutex locks |
| Lab 06 | Deadlocks, deadlock prevention and avoidance algorithms |
| Lab 07 | Memory management, paging, page replacement algorithms |
| Lab 08 | File systems, file allocation methods, disk scheduling algorithms |
| Lab 09 | System calls, shell scripting, and advanced OS concepts |
